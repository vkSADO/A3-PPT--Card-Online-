# PPT Online – Pedra Papel Tesoura (1v1)

Projeto acadêmico desenvolvido para a disciplina **Sistemas Distribuídos e Mobile**.

O **PPT Online** é um jogo multiplayer síncrono 1v1 baseado em uma variação estratégica de Pedra–Papel–Tesoura com cartas e blefe, implementado com arquitetura distribuída, backend em .NET e autenticação externa via Google.

O objetivo do projeto é demonstrar, de forma **simples e funcional**, a aplicação prática de conceitos de:
- Sistemas distribuídos
- Arquitetura baseada em serviços
- Comunicação em tempo real
- Consumo de API externa
- Persistência em nuvem
- Segurança por autenticação delegada

##  Tecnologias Utilizadas

- Godot 4.x (C#)
- ASP.NET Core
- SignalR
- Azure (App Service + banco gerenciado)
- Banco de dados NoSQL
- Autenticação via Google Identity (OAuth 2.0 / OpenID Connect)
